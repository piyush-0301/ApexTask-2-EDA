# ApexTask-2-EDA
Exploratory Data Analysis using Mall Customer Dataset
# 🛍️ Task 2 - Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a real-world customer dataset. The goal is to analyze customer behavior, understand spending patterns, and derive meaningful business insights using Python.

---

## 📂 Dataset Information

- **Dataset Name:** Mall Customer Dataset  
- **Total Records:** 200  
- **Total Features:** 5  

### 🔑 Features Description:

| Column Name | Description |
|------------|-------------|
| CustomerID | Unique ID of customer |
| Gender | Male / Female |
| Age | Age of customer |
| Annual Income (k$) | Yearly income in thousand dollars |
| Spending Score (1-100) | Score assigned based on spending behavior |

---

## 🎯 Objectives

- Perform data understanding and summary statistics  
- Analyze distribution of individual variables  
- Identify relationships between variables  
- Extract actionable insights from data  

---

## 🛠️ Tools & Technologies Used

- Python 🐍  
- Pandas  
- Matplotlib  
- Seaborn  
- Google Colab  

---

## 📊 Exploratory Data Analysis

### 🔹 1. Data Understanding
- Checked dataset structure using `.info()`  
- Generated statistical summary using `.describe()`  
- Analyzed categorical distribution using `.value_counts()`  

---

### 🔹 2. Univariate Analysis
- Distribution of Age  
- Distribution of Spending Score  
- Gender distribution  

📌 **Observation:**
- Most customers fall in the age group 20–40  
- Spending scores are widely distributed  

---

### 🔹 3. Bivariate Analysis
- Gender vs Spending Score  
- Age vs Spending Score  
- Income vs Spending Score  

📌 **Observation:**
- Female customers tend to have higher spending scores  
- Younger customers spend more  
- High income does not always mean high spending  

---

### 🔹 4. Data Visualization
- Histograms for distribution  
- Scatter plots for relationships  
- Color-based plots for category comparison  

---

## 📈 Key Insights

- 👩 Female customers show slightly higher spending behavior  
- 👦 Younger customers (20–35 age group) are high spenders  
- 💰 Customers with medium income often spend more than high-income customers  
- 📊 Spending behavior is not directly proportional to income
